# EKAM

EKAM is a modern event orchestration platform built for hackathons, case competitions, and judging workflows. It helps organizers manage events, participants, judges, submissions, approvals, and reports through a clean and interactive dashboard experience.

## Features

- Modern Landing Page
- Role-based Login & Signup
- Event Creation Dashboard
- Judge Evaluation Dashboard
- Team Formation Workflow
- Reports & Analytics UI
- Activity Tracking
- Dark/Light Theme Toggle
- Responsive Design
